

# NestJS Wiki Cool API

This project is a Node.js API built using the NestJS framework. It uses Wikipedia Featured Content API and offers translation for titles and extracts via the self-hosted LibreTranslate API.

## Functional Requirements

### Endpoints

#### /feed


#### /feed/translate/#language


## Getting Started

### Prerequisites

- Node.js (>= 20.x)
- npm (>= 10.x)

## Installation

## Running the app

Start app
```
npm run start
```

Start dev server
```
npm run start:dev
```

Start prod server
```
npm run start:prod
```

## Test

Tests
```
npm run test
```

E2E Tests
```
npm run test:e2e
```

Test Coverage
```
npm run test:cov
```
